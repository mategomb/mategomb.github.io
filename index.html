About

My name is Máté Gombkötő, and I'm a software developer from Budapest, Hungary.

I started off professionally on the front end of the web about three years ago and I'm one of those crazy few, who loves writing vanilla CSS without any library. But of course I'm no stranger to SASS/SCSS either.

I worked with JavaScript and Vue.js, and I'm also familiar with TypeScript. I'm also quite fond of using vanilla JS when it comes to building applications.

In the future, I would love to get more into server side and CLI/desktop application development as well with the Go programming language.
